# Problem Overview

Gig delivery workers depend on daily work availability.

External disruptions such as heavy rain, floods, severe air pollution, and extreme heat reduce deliveries and directly affect worker income.

These disruptions can reduce weekly earnings by up to 30%.

Current insurance models require manual claims and slow verification processes, making them unsuitable for gig workers.

This project proposes an automated parametric insurance platform that detects disruptions and compensates workers automatically.
